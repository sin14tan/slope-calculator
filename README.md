# 中山間地域等直接支払交付金 傾斜電卓

中山間地域等直接支払交付金の対象基準判定に必要な**勾配計算**を、現場で簡単に行えるWebアプリです。

## デモ

GitHub Pages: `https://<username>.github.io/slope-calculator/`

## 機能

### 基本計算
- A点・B点の標高と水平距離を入力するだけで勾配を自動計算
- 分数表示（1/n）と度数表示（°）を同時に算出
- 断面図と傾斜ゲージでビジュアルに判定結果を表示

### 傾斜区分の自動判定

#### 田（水田）— 分数表示で判定
| 区分 | 基準 | 色 |
|------|------|-----|
| 急傾斜 | 1/20 以上 | 🟠 オレンジ |
| 緩傾斜 | 1/100 以上 1/20 未満 | 🔵 青 |
| 対象外 | 1/100 未満 | ⚪ 灰色 |

#### 畑（草地・採草放牧地含む）— 度数表示で判定
| 区分 | 基準 | 色 |
|------|------|-----|
| 急傾斜 | 15度以上 | 🟠 オレンジ |
| 緩傾斜 | 8度以上 15度未満 | 🔵 青 |
| 対象外 | 8度未満 | ⚪ 灰色 |

### 端数処理
- **田**: 分母の小数第1位を**切り上げ**て判定
- **畑**: 度数の小数第1位を**切り捨て**て判定

### 加重平均計算
複数の測定単位が存在する場合、各測定単位の勾配（分母）と面積から加重平均を算出可能。

## 技術仕様

- **単一HTMLファイル**（`index.html` のみ）
- React 18 + Babel Standalone（CDN読み込み）
- SVGによるグラフィカル表示（断面図・傾斜ゲージ）
- サーバー不要・オフライン対応（初回読み込み後）
- モバイルファースト設計

3. Save → 数分で公開されます

```bash
git branch -M main
git remote add origin https://github.com/<username>/slope-calculator.git

```
